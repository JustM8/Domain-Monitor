# 🌐 Domain Monitor (Laravel)

## 📌 Опис

Веб-додаток для моніторингу доступності доменів з автоматичними перевірками, історією результатів та email-сповіщеннями.

Реалізовано як простий SaaS-подібний сервіс з можливістю керування доменами та налаштування перевірок.

---

## 🚀 Основний функціонал

### 🔐 Авторизація
- Реєстрація
- Вхід / вихід
- Захист приватних сторінок

### 🌍 Домени
- Додавання / редагування / видалення доменів
- Персональний список доменів користувача

### ⚙️ Налаштування перевірок
- Інтервал перевірки (для кожного домену)
- Таймаут запиту
- Метод перевірки (HTTP GET / HEAD)

### 🔁 Автоматичні перевірки
- Запуск через Laravel Scheduler
- Перевірка доменів за заданим інтервалом
- Підтримка глобального fallback інтервалу (Settings)

### 📊 Історія перевірок
- Дата перевірки
- HTTP статус
- Час відповіді
- Помилки (якщо є)

### 📧 Email-сповіщення
- Надсилаються при падінні доменів (FAIL)
- Email задається у Settings

---

## 🛠 Технології

- Laravel 9+
- MySQL
- Bootstrap (UI)
- cURL (перевірка доменів)
- Laravel Scheduler
